# Level 1 · Entry <span class="level-badge">Foundations</span>

Goal: get comfortable with the AWS CLI and console, understand the core
services almost every architecture is built on, and ship a small end-to-end
serverless web app.

## Modules

1. [Setup & AWS CLI](01-setup-cli.md)
2. [IAM & Security Basics](02-iam-security-basics.md)
3. [EC2 Compute](03-ec2-compute.md)
4. [S3 Storage](04-s3-storage.md)
5. [VPC Networking Basics](05-vpc-networking-basics.md)
6. [RDS Databases](06-rds-databases.md)
7. [Lambda & Serverless](07-lambda-serverless.md)
8. [CloudWatch Monitoring & Logging](08-cloudwatch-monitoring.md)
9. [CloudFormation & IaC Intro](09-cloudformation-iac.md)
10. [Capstone — Serverless Web App](10-capstone-project.md)

By the end of this level you'll be able to provision compute, storage,
networking, and databases from the CLI, secure them with IAM, monitor them
with CloudWatch, describe them as code with CloudFormation, and deploy a
small serverless web app — and tear it all back down cleanly so it doesn't
keep costing you money.
