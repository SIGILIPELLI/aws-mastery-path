# AWS Mastery Path

A free, structured, module-wise AWS training program — entry level to master level, with runnable CLI commands and real configuration in every module, and a hands-on project at the end of each level.

**Live site:** https://sigilipelli.github.io/aws-mastery-path/

## Contents

- **Level 1 · Entry** — account setup, IAM, EC2, S3, VPC networking basics, RDS, Lambda/serverless, CloudWatch, CloudFormation, capstone (serverless web app)
- **Level 2 · Intermediate** — containers (ECS/Fargate), auto scaling & load balancing, Route 53, SNS/SQS, DynamoDB, Elastic Beanstalk, Secrets Manager, CloudFront, cost management
- **Level 3 · Advanced** — advanced VPC, EKS/Kubernetes, Step Functions, advanced IAM, Terraform, CI/CD, multi-region DR, X-Ray, security deep dive
- **Level 4 · Master** — Well-Architected Framework, advanced serverless, data lakes & analytics, EKS at scale, multi-account strategy, cost optimization, advanced networking, compliance & governance, chaos engineering, capstone

## Local development

```bash
python3 -m venv .venv
.venv/bin/pip install mkdocs-material
.venv/bin/python -m mkdocs serve
```
